## Project Description
AiFindr is a full-stack web application designed to help users discover, compare, and manage AI tools through a centralized platform.

## Features
- AI tool search and filtering
- Tool comparison
- User ratings and bookmarking
- Admin dashboard for content management

## Technologies Used
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: PHP
- Database: MySQL

## Project Architecture
- Follows SDLC approach: Requirement Analysis, Design, Development, Testing, Deployment

## How to Run the Project
1. Install XAMPP
2. Import the MySQL database
3. Place project folder inside `htdocs`
4. Start Apache and MySQL
5. Open browser and navigate to `localhost/AiFindr`
